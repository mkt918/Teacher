# 担任業務支援アプリ

**🔗 デモ・利用はこちら: [https://mkt918.github.io/Teacher/](https://mkt918.github.io/Teacher/)**

担任業務を効率化するための統合Webアプリケーションです。生徒名簿管理、個人メモ、席替え、保護者会調整など、日々の業務をサポートします。

## 🎯 主な機能

### ✅ 実装済み（フェーズ1〜2）

- **📚 ダッシュボード**: 各種ツールへの統合アクセス
- **👥 生徒名簿管理**:
  - 生徒の追加・編集・削除
  - 番号順・五十音順ソート
  - CSV一括インポート/エクスポート
  - 検索・フィルター機能
- **📝 個人メモ（Obsidian風）**:
  - 生徒ごとの個別メモ
  - Markdown記法サポート
  - タグ機能（#面談、#保護者連絡など）
  - 全メモのテキスト一括出力（NotebookLM連携用）
- **🪑 席替えツール**:
  - くじ引きモード（トランプ／番号札）
  - 座席の自動配置・入れ替え
  - 印刷レイアウト調整
- **📅 保護者会調整ツール**:
  - 希望調査集計
  - 自動スケジュール調整
  - 調整結果の印刷
- **📅 時間割管理（NEW）**:
  - 日付による曜日自動表示
  - クラス時間割／自分の時間割の切り替え
  - ドラッグ＆ドロップによる直感的な時間割編集
  - 変更箇所の強調表示とメモ機能（自分／クラス別）
  - 科目別授業時数の集計
- **💾 保存システム**:
  - リアルタイムセーブ（3スロット、自動保存）
  - ステートセーブ（3スロット、手動保存）
  - LocalStorage使用（ブラウザローカル保存）

### 🚧 今後の拡張予定

- 📋 クラス係掲示ツール
- データのクラウド同期（Google Drive連携など）

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. または、ローカルサーバーで起動:

   ```powershell
   # Python 3の場合
   python -m http.server 8000
   
   # ブラウザで http://localhost:8000 にアクセス
   ```

### 基本的な使い方

1. **生徒名簿の登録**
   - 「生徒名簿」ページで「➕ 生徒追加」ボタンをクリック
   - 番号（4桁）、名前（漢字）、名前（ふりがな）を入力
   - CSV一括読込も可能

2. **個人メモの記録**
   - 「個人メモ」ページで生徒を選択
   - Markdown形式でメモを記入
   - タグ（#面談など）を使って分類
   - 「📤 全メモ出力」でテキストファイルとしてエクスポート

3. **時間割の管理**
   - 「各種設定」→「時間割」でベースの時間割を設定
   - ドラッグ＆ドロップで科目を配置
   - ダッシュボードで日々の変更やメモを入力

4. **データの保存**
   - **リアルタイムセーブ**: データ変更後、1秒で自動保存
   - **ステートセーブ**: フッターの「💾 ステート保存」で手動保存
   - 設定画面から保存履歴を確認・管理

## 💾 データ保存について

### LocalStorageの特性

- **保存期間**: 基本的に半永久的（ブラウザを閉じても残ります）
- **注意点**:
  - ブラウザのキャッシュ・履歴を削除すると消えます
  - 異なるブラウザやデバイス間では共有されません
  - プライベートモード（シークレットモード）では保存されません

### 推奨運用

1. 定期的にステートセーブを実行
2. 重要なデータはCSV/テキスト出力でバックアップ
3. 同じブラウザを使い続ける
